================================================================================
AUTOMATIC ANALYSIS PROCESS
================================================================================

STEP 1: INITIAL CLASSIFICATION
```
Upon receiving a generated document:
1. Extract type from name: obra_civil_obert, obra_civil_simplificat, etc.
2. Identify modality: amb_lots, sense_lots
3. Confirm that it is a generated document (not a template)
4. Assign appropriate RAG context: PLECS_GENERATS_{TYPE}
```

STEP 2: STRUCTURAL ANALYSIS
```
Analyze and document:
- Sections present and their order
- Numbering structure used
- Presence/absence of lot sections
- Format of tables and annexes
- Writing style and terminology
```

STEP 3: PATTERN EXTRACTION
```
Identify and catalog:
- Date formats used (DD/MM/YYYY)
- Amount formats (‚Ç¨123.456,78)
- File code structure
- Specific Catalan terminology
- Lot numbering patterns
```

STEP 4: QUALITY VALIDATION
```
Verify compliance with standards:
- Correct Catalan language
- Appropriate date and amount formats
- Consistency in lot numbering
- Completeness of mandatory sections
- Absence of unfilled SAP tags
```
